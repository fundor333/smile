# emojismile

smile enoji plugin for oh-my-zsh.

**Maintainers**: [fundor333](https://github.com/fundor333) 

To use it add `smile` to the plugins array in you zshrc file.

```zsh
plugins=(... smile)
```

## Usage

| Command | Description                     |
| --------| ------------------------------- |
| `smile` | Print random smile         |

Example: output of `smile`:

```
(°,,,°)
```
